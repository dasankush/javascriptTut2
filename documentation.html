Documentation
<Add Person Count>

The <Add> button increments the person count by +1.

<Remove Person Count>

The <Remove> button decrements the person count by +1. Minimum person count is 0.

<Reset Person Count>

The <Reset> button resets the person count i.e. count = 0.

<Save Person Logs>

The <Save> button saves the person logs and displays below. Current log should not be equal to previous log.

<Clear Person Logs>

The <Clear> button deletes the person count logs.
